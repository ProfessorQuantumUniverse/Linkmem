### Speichere und sortiere deine Links – einfach und nach Projekten geordnet.

Ein schlankes Desktop-Tool, mit dem du Links sammelst und projektweise organisierst – Schluss mit dem Chaos aus verstreuten Lesezeichen.

## ✨ Features

- 💾 Links speichern & wiederfinden
- 🗂️ Ordnung **nach Projekten**
- 🖥️ Verfügbar für Windows
- 🐍 In Python geschrieben

## 🛠️ Tech-Stack

`Python` · `Tkinter`

## 🚀 Starten

```bash
python modern_link_manager.py
```

> 💡 *Tipp:* Die fertige `.exe` gehört eigentlich in die **Releases** statt ins Repo – das hält das Repo schlank.
